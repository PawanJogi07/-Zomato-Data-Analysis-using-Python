# 🍽️ Zomato Data Analysis using Python

## 📌 Project Overview

This project focuses on analyzing Zomato restaurant data to uncover meaningful business insights through Exploratory Data Analysis (EDA).

The analysis helps understand restaurant ratings, online delivery trends, country-wise distribution, cuisines, and customer preferences.

## 🚀 Objectives

* Perform Data Cleaning

* Analyze Restaurant Ratings
* Explore Country-wise Restaurant Distribution
* Study Online Delivery Trends

* Most restaurants belong to a few major countries.
* Online delivery services are concentrated in selected regions.
* Higher-rated restaurants receive significantly more customer votes.
* Certain cuisines dominate the platform.
* Restaurant ratings show a strong concentration in mid-to-high rating ranges.
